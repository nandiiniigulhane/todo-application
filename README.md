# 📝 Todo Application

A full-stack Todo Application built with **React, TypeScript, and Spring Boot**.
It allows users to create, update, view, and delete tasks with a modern and clean user interface.

---

## 🚀 Tech Stack

### Frontend

- React
- TypeScript
- Vite
- CSS

### Backend

- Java
- Spring Boot
- REST API

---

## 🛠️ Installation & Running the Project

### 1. Clone the Repository

```bash
git clone https://github.com/nandiiniigulhane/todo-application.git
cd todo-application
```

---

### 2. Run Backend

```bash
mvn spring-boot:run
```

Backend will start on:

```
http://localhost:8080
```

---

### 3. Run Frontend

```bash
cd todoapp_frontend
npm install
npm run dev
```

Frontend will run on:

```
http://localhost:5173
```

Open it in your browser.

---

## 📌 API Connection

The frontend connects to the backend running at:

```
http://localhost:8080
```

Make sure the backend is running before starting the frontend.
